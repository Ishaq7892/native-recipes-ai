# native-recipes-ai
Backend Setup (Python Server)
1. Install Python 3.10 or higher.
2. Create a virtual environment: python -m venv venv.
3. Activate the environment.
4. Install dependencies using: pip install -r requirements.txt.
5. Run the FastAPI server using: uvicorn app.main:app --reload.
6. Backend handles WebSocket communication, speech-to-text processing, AI processing, and
text-to-speech response generation.



Frontend Setup (React Project)
1. Install Node.js and npm.
2. Create a React project using Create React App or Vite.
3. Install dependencies using npm install.
4. Organize project folders (components, pages, services).
5. Use Axios or Fetch to communicate with backend APIs.
6. Start development server using npm start.
7. Frontend handles user input, displays recipe results, and communicates with the backend AI
services.
